About Us

The Toy Problem App is an open source project, created by Rory Campbell and Dave Mun.

The Toy Problem App was created to help people put their JavaScript skills to the test. Having been used to daily JavaScript challenges at Hack Reactor, we wanted to share the learning experience and to encourage others.

If you would like to contribute to the project, please make a pull request to our GitHub repo. Requests for new features or bug fixes are also welcome.

Many thanks to the various open source technologies that helped us to build this project including AngularJS, Express, and Node.
